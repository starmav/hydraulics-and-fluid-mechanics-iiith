


INSTRUCTIONS ON HOW TO USE SIMULATOR:
->Firstly run the Index.html file in the browser.
->Click on the Start Button to start the simulation.
->Click on Start Button to restart the simulation.
->Click on Calculations Button to calculate Coefficient Cd.
	|->Give the required values to calculate Cd.
	   (The values should be only positive integers less than 10,000).
	|->Click on Submit button to calculate Cd.
	|->Click on Close Button to close the Calculations model or window.
